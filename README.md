Grocery Store Management System – 8086 Assembly

A complete console-based e-commerce and inventory management system developed in 8086 Assembly Language using EMU8086. The project simulates a real grocery store environment with customer shopping, cart management, discount/payment processing, product searching, customer rating, and a PIN-protected admin panel.

Features
Product browsing by category
Add to cart & remove last item
Dynamic bill calculation
Discount code validation system
Cash & Cash on Delivery payment methods
Product search and filtering
Customer review & rating system (1–5)
Admin panel with PIN authentication
Add, delete, and update products
Multi-digit number printing using stack/division logic
Array-based cart and inventory management
Technologies Used
8086 Assembly Language
EMU8086
DOS Interrupts (INT 21H)
Core Concepts Implemented
Arrays & memory management
Loops and conditional branching
Stack operations
String handling
Procedure-based modular programming
User input validation
Arithmetic and division operations
Admin Access

Default PIN: 1234

Learning Outcomes

This project demonstrates low-level programming concepts including memory manipulation, interrupt handling, modular assembly procedures, and real-world system simulation using 8086 Assembly.
